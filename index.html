Explorando Laços for em Diferentes Linguagens

JavaScript

for (let i = 0; i < 5; i++) {
    console.log("Iteração:", i);
}

Python

for i in range(5):
    print("Iteração:", i)

PHP

for ($i = 0; $i < 5; $i++) {
    echo "Iteração: " . $i . "\n";
}

C++

for (int i = 0; i < 5; i++) {
    cout << "Iteração: " << i << endl;
}

Comparação

O laço for é uma estrutura comum em muitas linguagens de programação, mas com algumas diferenças:

  • JavaScript, PHP e C++ usam uma sintaxe similar com três partes: inicialização, condição e incremento.
  • Python usa uma abordagem mais simples com range() para gerar sequências.
  • C++ requer declaração de tipos explícita (int i).
  • PHP usa o símbolo $ para variáveis.
